# HarmonyStream

A modern music streaming web application built with Next.js 14, TypeScript, and Tailwind CSS.

## Overview

HarmonyStream is a sleek, responsive music player application that delivers a premium listening experience. The application features a modern user interface inspired by leading music streaming platforms, complete with full audio playback controls, playlist management, and theme customization.

---

## Features

| Feature | Description |
|---------|-------------|
| **Audio Playback** | Full-featured music player with play, pause, skip, shuffle, and repeat controls |
| **Search** | Real-time search functionality across songs, artists, and albums |
| **Playlists** | Create, manage, and organize custom playlists |
| **Liked Songs** | Save and access favorite tracks quickly |
| **Theme Toggle** | Seamless switching between dark and light modes |
| **Responsive Design** | Optimized experience across desktop, tablet, and mobile devices |
| **Now Playing** | Dynamic display of current track with album artwork |
| **Queue Management** | View upcoming tracks and playback history |

## Quick Start

**Prerequisites:** Node.js 18+ and npm

```bash
# Clone the repository
git clone https://github.com/Maviyanoor/nextjs-music-app.git

# Navigate to project directory
cd nextjs-music-app

# Install dependencies
npm install

# Start development server
npm run dev
```

Open **http://localhost:3000** to view the application.
